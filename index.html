1

安装 CLI

需要 Node.js >= 18 与 npm。

npm install -g @z_ai/zai-cli
2

配置 API Key

从 Z.ai 控制台获取 Key，选择对应区域。

zai-cli auth set "your-api-key" --region china
3

开始调用

运行第一条命令，验证安装成功。

zai-cli image-gen "卡通的爱因斯坦"
