## 目标                                                                                                                                   
                                                                                                                                             
   在仓库里实现一个**网页版 macOS 桌面环境**——在浏览器里还原 macOS Big Sur / Monterey 的桌面体验，可作为该项目的展示首页或独立 demo。        
                                                                                                                                             
   ## 功能要求                                                                                                                               
                                                                                                                                             
   ### 核心桌面                                                                                                                              
   - 顶部菜单栏（Apple logo + 当前应用名 + 系统托盘：时间、电池、WiFi）                                                                      
   - 桌面区域：支持放置图标/窗口，有渐变壁纸                                                                                                 
   - 底部 Dock 栏：应用图标、鼠标悬停放大效果、点击启动窗口                                                                                  
                                                                                                                                             
   ### 窗口系统                                                                                                                              
   - 可拖拽移动的窗口（标题栏 + 内容区）                                                                                                     
   - 窗口可最小化（缩回 Dock）、关闭、全屏                                                                                                   
   - 窗口聚焦时其他窗口变暗（z-index 管理）                                                                                                  
   - 窗口打开/关闭动画                                                                                                                       
                                                                                                                                             
   ### 至少 3 个内置应用                                                                                                                     
   1. **访达 (Finder)** —— 简单的文件列表面板                                                                                                
   2. **终端 (Terminal)** —— 一个可输入命令的模拟终端（echo 即可）                                                                           
   3. **关于本机 (About)** —— 系统信息面板                                                                                                   
                                                                                                                                             
   ### 视觉细节                                                                                                                              
   - macOS 风格的圆角、毛玻璃（backdrop-blur）、阴影                                                                                         
   - 暗色/亮色主题切换                                                                                                                       
                                                                                                                                             
   ## 技术约束                                                                                                                               
                                                                                                                                             
   - 纯前端实现（HTML/CSS/JS，可用 vanilla 或现有框架）                                                                                      
   - 不依赖后端服务                                                                                                                          
   - 响应式：在窄屏上也能基本可用                                                                                                            
                                                                                                                                             
   ## 说明                                                                                                                                   
                                                                                                                                             
   这是一个较大的任务，请拆分成多个可独立交付的 PR（比如：桌面骨架 → 窗口系统 → Dock → 各个应用 → 主题切换）。每个 PR 聚焦一个子模块。
